# BCA CSV & PDF ke Excel

Modul ini berbasis python bertujuan untuk merubah dokumen mutasi BCA jenis csv & PDF ke Excel

## Persyaratan

Lewati langkah 1—2 jika peranti Anda telah terpasang python, pip, pandas, openpyxl dan pdfplumber 
- Pasang [python](https://www.python.org/downloads/) 
- Pasang pandas & openpyxl dengan perintah "pip install pandas openpyxl pdfplumber"
- Unduh module ini dan ekstrak
- Buat folder baru dan letakkan modul yang telah di unduh pada folder tersebut 
- bcacsv2excel.py (untuk konversi csv ke excel)
- bcacsv2exceldbcr.py (untuk konversi csv ke excel dan memisahkan nominal debit/credit)
- bca_extract_pdf.py (untuk konversi mutasi jenis PDF ke excel)
- Salin semua data mutasi BCA jenis csv ke folder tersebut 
- Klik 2× modul
____

Telah di uji pada Windows 11 & Ubuntu 24.04
